# PerfectProof

This repository contains a blank C# solution and project named `PerfectProof`.

## Structure

- `PerfectProof.sln` - solution file
- `PerfectProof/PerfectProof.csproj` - class library project
- `PerfectProof/BigDecimalNumber.cs` - big decimal number implementation

## Build

```bash
dotnet build PerfectProof.sln
```
